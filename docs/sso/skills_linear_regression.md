### Correlation
#### Testing for linear correlation with the $p$-value method
**Hypothesis**  
$H_0: \rho=0$  
$H_1: \rho \neq 0$

**Test statistic**

$$t = \frac{r}{\sqrt{\frac{1-r^2}{n-2}}}$$


**Procedure**  
Two-sided $t$ test with $df = n-2$.
